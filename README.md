## Structure

`index.html` 1ファイル（CSS・JS・アイコン画像を内包）の静的サイトです。ビルド不要で、push するだけで GitHub Pages により公開されます。

## Editing Notes

* **配色** — `index.html` 冒頭 `:root` の CSS 変数（`--bg`, `--accent` など）を変更すると全体に反映されます
* **言語** — 日英切替対応。`class="ja"` / `class="en"` の要素が右上のトグルで切り替わります
* **項目の追加** — `.entry` ブロックをコピーして編集します。非公開の下書きは `hidden` 属性で隠せます（HTML内にテンプレートをコメントで同梱）
